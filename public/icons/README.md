# AgroRuta Huánuco — Íconos PWA

Los íconos se generan con `vite-plugin-pwa` en build.
Para desarrollo, agrega íconos PNG en esta carpeta:

- icon-192.png  (192×192)
- icon-512.png  (512×512)
- apple-touch-icon.png (180×180)

Puedes generarlos con herramientas como:
- https://realfavicongenerator.net/
- https://maskable.app/
